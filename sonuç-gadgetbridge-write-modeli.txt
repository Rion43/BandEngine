SDK 33+ gatt.writeCharacteristic() -> onCharacteristicWrite callback'ini tetikler.
Callback yazma Bluetooth stack'e queue'ya alindiginda gelir (karşı taraf gordugunde degil).

Web Bluetooth writeValueWithoutResponse():
  Promise resolve = veri Bluetooth stack'e gonderildiginde
  Android callback ile ayni anlamda

FARK YOK.

9. SPP ACK Gelmeden Sonraki Write
──────────────────────────────────
Gadgetbridge SPP ACK'i beklemez.
TransactionBuilder icindeki tum WriteAction'lar BLE callback seviyesinde siralanir:
  1. writeCharacteristic() -> onCharacteristicWrite callback
  2. callback gelince latch cozulur
  3. sonraki action baslar

SPP ACK notification ayri bir kanaldan (notify characteristic) gelir.
BLE write sirasi ile SPP ACK arasinda BAGLANTI YOK.

10. Band 9 Icin Ozel Write Queue
─────────────────────────────────
XiaomiBleProtocolV2.java: Ozel write queue YOK.
Standart TransactionBuilder.writeChunkedData() + WriteAction kullanir.
Herhangi bir ozel queue, buffer, veya retry mekanizmasi yok.

OZET: Web Bluetooth vs Android Gadgetbridge
======================================================================

Xiaomi Smart Band 9 icin BLE write modeli:

                  | Android Gadgetbridge      | Web Bluetooth (BandEngine)
                  |---------------------------|---------------------------
Write Type        | WRITE_TYPE_NO_RESPONSE    | writeValueWithoutResponse()
Write Method      | writeCharacteristic()     | writeValueWithoutResponse()
Callback/Signal   | onCharacteristicWrite     | Promise.resolve()
Callback Anlami   | queue'ya alindi           | queue'ya alindi
Write Sirasi      | latch ile sirali          | await ile sirali
SPP ACK Bekleme   | beklemez                  | beklemez
Ozel Queue        | yok                       | yok

Sonuc: Android ile Web Bluetooth write modeli arasinda
Xiaomi Band 9 icin HICBIR FARK YOKTUR.

Baglanti kopmasinin nedeni write modeli DEGILDIR.
Muhtemel nedenler (elenmemis):
  - Web Crypto AES-CTR ile Android AES/CTR/NoPadding arasinda fark
  - protobuf encoding farki
  - Band'in beklentisi: post-auth paketleri sayisi/sirasi
  - Bluefy/iOS Web Bluetooth stack'inde BLE supervision timeout
